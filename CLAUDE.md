# Proyecto 02 — YarigTelegram

> Control completo de Yarig.ai desde Telegram con panel interactivo de tareas, resumenes automaticos y arranque diario de mision.

## Contexto

Replica la operativa diaria de Yarig.ai dentro de Telegram para gestionar tareas, pausarlas, finalizarlas y consultar resumenes sin salir de la app. Integra el consejo AdmiraNext via LLM (modelo llama gratuito) con puerto API dedicado.

## 1 2 3 basico

1. leer `onboarding` y estado actual del bot;
2. avanzar sin romper el panel inline de Telegram;
3. cerrar con bot funcionando y URL publica verificable.
